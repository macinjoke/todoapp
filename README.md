# todoapp

![タイトル](https://raw.github.com/wiki/tonkatu05/todoapp/images/todo_app.png)

React 練習用のTODOアプリを1から作成。 `create-react-app` を使用。

## ENVIRONMENT
- python 3.6.0
- npm 3.10.10

## SETUP

```
npm install
pip install -r requirements.txt
```

## RUN

```
python server.py
npm start 
```
